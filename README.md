# Проект YaMDb

## Описание

Проект **YaMDb** — это сервис, который собирает отзывы пользователей на различные произведения. Сами произведения (фильмы, книги, музыка) в сервисе не хранятся, YaMDb — это площадка для обсуждения и оценки.

## Задача проекта

Сервис позволяет пользователям ориентироваться в огромном количестве контента благодаря системе рейтингов и честных отзывов.

## Основные возможности:

* **Рейтинги:** На основе оценок пользователей (от 1 до 10) формируется усредненный рейтинг произведений.

* **Категории и жанры:** Произведения делятся на типы (книги, кино, музыка) и объединяются по жанрам.

* **Сообщество:** Аутентифицированные пользователи могут оставлять отзывы и комментировать чужие публикации.

* **Гибкое управление:** Система ролей (автор, модератор, администратор) обеспечивает порядок и модерацию контента.

## Установка

Инструкция для развертывания проекта на локальной машине:

**1. Клонировать репозиторий и перейти в него в командной строке:**

```
git clone git@github.com:IamAntonion/api-yamdb.git
cd api_yamdb
```

**2. Cоздать и активировать виртуальное окружение:**

Для Windows:

```
python -m venv venv
source venv/Scripts/activate
```

Для Linux/macOS:

```
python3 -m venv venv
source venv/bin/activate
```

**3. Установить зависимости из файла requirements.txt:**

```
python -m pip install --upgrade pip
pip install -r requirements.txt
```

**4. Выполнить миграции:**

```
python manage.py migrate
```

**5. Запустить проект:**

```
python manage.py runserver
```

Проект будет доступен по адресу: http://127.0.0.1:8000/

Документация к API (ReDoc) доступна после запуска проекта по адресу: http://127.0.0.1:8000/redoc/

## Примеры использования API

**Регистрация нового пользователя**

```
POST /api/v1/auth/signup/
```
Тело запроса (JSON):
```
{
  "email": "user@example.com",
  "username": "new_user"
}
```

**Получение JWT-токена**

```
POST /api/v1/auth/token/
```
Тело запроса (JSON):
```
{
  "username": "new_user",
  "confirmation_code": "код_из_письма"
}
```

**Добавление отзыва**

```
POST /api/v1/titles/{title_id}/reviews/
```
Тело запроса (JSON):
```
{
  "text": "Потрясающее произведение, рекомендую всем!",
  "score": 10
}
```

**Получение списка произведений**

```
GET /api/v1/titles/
```
